# Fake News Detection using Social Media Data

TCS iON – Industry Project

## Overview
This project implements a Fake News Detection system using Natural Language Processing
and Machine Learning techniques. The system classifies social media-style news as Fake or
Real along with a confidence score. Low-confidence predictions are handled as inconclusive.

## Tech Stack
- Python
- Streamlit
- Scikit-learn
- TF-IDF
- Logistic Regression

## Project Structure
- `code/` – Streamlit app and trained model files
- `documents/` – Project report and test documents
- `demo/` – Demo video link
